BUILD:

Press Ctrl+Shift+P (or Cmd+Shift+P on Mac), type "Tasks: Run Task", and select:

build - Builds the project in Debug configuration
clean - Cleans the build outputs
publish-release - Publishes the project in Release configuration (includes clean)

-da lauch Debug MCP Server (fa build + run perchè esegue la dll)
